# synent_task3_netflix_EDA
# 🎬 Netflix Exploratory Data Analysis (EDA) Project

## 📌 Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on the Netflix dataset to uncover trends, patterns, and insights about the content available on the platform.

---

## 🎯 Objective

To analyze the dataset and identify meaningful patterns using statistical analysis and visualizations.

---

## 🏢 Internship

This project was completed as part of an internship at **Synent Technologies**, where I gained hands-on experience in data analysis and visualization using real-world datasets.

---

## 📂 Dataset

* **Name:** Netflix Dataset
* **Source:** Kaggle
* Contains information about movies and TV shows such as title, type, country, release year, rating, and genres.

---

## 🛠️ Tasks Performed

### ✅ 1. Summary Statistics

* Generated descriptive statistics for numerical data
* Analyzed distributions and key metrics

### ✅ 2. Data Cleaning

* Handled missing values
* Removed duplicate records
* Converted date columns into proper datetime format

### ✅ 3. Correlation Analysis

* Identified relationships between numerical features
* Visualized correlations using heatmaps

### ✅ 4. Trend Identification

* Analyzed content growth over the years
* Compared Movies vs TV Shows
* Identified top countries and popular genres

---

## 🧰 Technologies Used

* Python 🐍

* Matplotlib
* Seaborn

---

## 📊 Output
